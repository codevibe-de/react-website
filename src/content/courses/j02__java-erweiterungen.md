## Description

Mit diesem Seminar bauen Sie Ihr Java Wissen um weiterführende Konzepte aus.

Anhand zahlreicher Übungsprojekte und Katas werden neue Konzepte eingeführt und nachhaltig gelernt.

## Goal

Verständnis und eigenständige Entwicklung von Anwendungen, die von komplexeren Konzepten der Sprache Gebrauch machen.

## Target Audience

Software-Entwickler:innen mit Java-Kenntnissen, z.B. aus dem Seminar Java Grundlagen.

## Outline

1. Generische Datentypen
2. Reflection
3. weitere Klassenarten
4. Sortieren und Suchen
5. Streams
6. Lambdas
7. Multi-Threading
8. Java I/O
